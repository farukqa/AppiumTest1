Upon initial cloning of this repository, do the following:
- open a command line at the projects root
- execute: "git config --local include.path ../.gitconfig"
- execute: "git init-ignores"
- execute: "git config --global --add user.name <your user (displayed)name for Bitbucket>"
- execute: "git config --global --add user.email <your email address for Bitbucket>"
- execute: "git this-user", and verify the names set above
